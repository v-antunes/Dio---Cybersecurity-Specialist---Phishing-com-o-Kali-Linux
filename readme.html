Entrega do Projeto de Phishing com Uso do Kali Linux Introdução: Descubra as várias maneiras de utilizar o Kali Linux (VM, Cloud, Container, Smartphone). Este projeto visa criar um método de captura de dados por meio de uma máquina virtual (Proxmox), utilizando uma VM executando a última versão do Kali Linux. Etapas do Projeto: 1. Configuração Inicial: Baixe o Proxmox e realize a instalação no seu computador. Após a instalação, configure a máquina como Kali Linux e faça a atualização do sistema. 2. Configuração do Setoolkit: Abra o Setoolkit e insira sua senha de root. Escolha as opções: 1 (Social-Engineering Attacks) 2 (Website Attack Vectors) 3 (Credential Harvester Attack Method). 3. Você terá opções como usar um template de site pronto (por exemplo, tela de login do Google), clonar um site ou fazer uma importação customizada. Neste caso, escolha a opção 1 (Web Templates). Indique o endereço IP para o servidor de captura de dados. Nesse exercício, utilize o IP padrão fornecido. 4. Seleção do Site de Login: Escolha o site de login do Google (opção 2). Agora, o site que captura os dados está criado. Acesse o IP informado acima para realizar testes. 5. Testes e Observações: Observe que todos os dados inseridos nos campos de senha e login são capturados no Kali Linux. Conclusão: Este projeto oferece uma abordagem prática para entender o uso do Kali Linux na criação de um ambiente de phishing. Certifique-se de realizar apenas atividades éticas e respeitar a privacidade e segurança online.
